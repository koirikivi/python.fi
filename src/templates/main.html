Tervetuloa

Python Suomi on vielä rekisteröitymisvaiheessa oleva yhdistys, joka parantaa Python-ohjelmointikielen tuntemusta Suomessa sekä edistää alan ihmisten keskinäistä yhteistyötä ja verkostoitumista.

Uutisia

Yhdistys perustettu

Perustamiskokous järjestettiin 7.5.2011 Espoossa. Kokouksessa yhdistyksen nimeksi äänestettiin Python Suomi, käytiin läpi ja hyväksyttiin yhdistyksen säännöt, valittiin ensimmäinen hallitus sekä päätettiin jäsenmaksuista.

Kiitos kaikille osallistuneille!

Yhdistyksen perustamiskokous

Python Interest Group Finland ry:n perustamiskokous järjestetään lauantaina 7.5.2011 klo 10:00 DjangoCon Finlandin yhteydessä Aalto Venture Garagen tiloissa, Betonimiehenkuja 3, Espoo. Perustamiskokouksessa mm. hiotaan kuntoon yhdistyksen säännöt (työversio Google Docsissa) sekä valitaan yhdistyksen ensimmäinen hallitus. Kaikki ovat tervetulleita!

Virallinen kokouskutsu yhdistyksen perustamiskokoukseen löytyy täältä.

Kehittäjä hakusessa
